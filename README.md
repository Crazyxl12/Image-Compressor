# Compressor de Imagens

Projeto simples de site web para comprimir imagens no próprio navegador.

## Como usar

Abra o arquivo `index.html` no navegador, escolha até 10 imagens, ajuste a redução desejada e clique em **Comprimir imagens**.

A redução desejada funciona como meta: 10% tenta deixar o arquivo cerca de 10% menor, 50% tenta deixar cerca de 50% menor, e assim por diante. Dependendo da imagem, o resultado pode não bater exatamente a meta.

Você pode selecionar várias imagens com Ctrl/Shift ou escolher uma imagem por vez. Cada nova escolha entra na lista de imagens selecionadas.

Quando apenas uma imagem é comprimida, o site mostra a prévia original e comprimida. Quando várias imagens são comprimidas, o site mostra a lista dos resultados e permite baixar tudo comprimido em um único arquivo.

O site aceita JPG, PNG e WEBP. As imagens não são enviadas para nenhum servidor.
